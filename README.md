# Mexican Dishes

A simple recipe website built solely with HTML, featuring recipes for three classic Mexican dishes that have been Americanized beyond what my family would forgive.

[Live Demo](https://ian-vazquez.github.io/odin-recipes/)

## About

There are four pages to the website. First is the homepage, which houses links to the three recipe webpages: tacos, enchiladas, chilaquiles. Each recipe webpage has a photo and description, ingredient list, recipe steps, and a link back to the homepage.

## How to view

Clone the github repo and open `index.html` in your browser.
```
git clone https://github.com/ian-vazquez/odin-recipes.git
```
## What I learned

- Creating and structuring an HTML document (Hyperlinks, lists, images, headings, etc.)
- Employing both URLs and relative paths to link between pages
- Git Workflow essentials: adding, committing, and pushing changes to GitHub; deploying with GitHub Pages
